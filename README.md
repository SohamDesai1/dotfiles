# Dotfiles

My personal dotfiles for Arch Linux with Hyprland.

## Installation

```bash
./install.sh
```

## Dev Tools

To install dev tools run:

```bash
./install_dev.sh
```

## Contents

- `.bashrc`, `.bash_profile`, `.profile` - Shell configs
- `.zshrc` - Zsh configuration with Oh My Zsh
- `.p10k.zsh` - Powerlevel10k theme
- `config/hypr/` - Hyprland (Wayland compositor) config
- `config/nvim/` - Neovim config
- `config/fish/` - Fish shell config
- `config/kitty/` - Kitty terminal config
- `config/waybar/` - Waybar config
- `config/wofi/` - Wofi (app launcher) config
- `config/btop/` - Btop config
